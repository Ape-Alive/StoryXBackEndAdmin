## 文档结构

每个接口文档包含以下部分：

1. **summary**: 接口简要说明
2. **description**: 详细描述
   - 流程说明
   - 字段说明
   - 注意事项
   - 使用示例（curl）
3. **tags**: 接口分类标签
4. **security**: 认证要求
5. **parameters/requestBody**: 请求参数
   - 类型、格式、示例
   - 必填/可选说明
   - 取值范围和验证规则
6. **responses**: 响应说明
   - 成功响应（200）
     - 完整的数据结构
     - 字段说明和示例
   - 错误响应（400/401/403/404）
     - 错误场景示例
     - 错误消息说明
